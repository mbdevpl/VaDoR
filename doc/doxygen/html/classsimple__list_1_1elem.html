Inserts new element after this element.

If this element is empty, it does nothing unless the element is connected to a list which is empty.

If this element is not empty, and is connected to any list, the the changes in the list are handled by this method:

  • length of list increases
  • if element is inserted after the last one, the last_elem moves

Inserts a new element before this element.

If this element is empty, it does nothing unless the element is connected to a list which is empty.

If this element is not empty, and is connected to any list, the the changes in the list are handled by this method:
